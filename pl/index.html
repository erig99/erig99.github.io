XUnit

Stosowanie zasad Domain-Driven Design do projektowania czystej, modułowej architektury skupionej na logice biznesowej, szczególnie w środowiskach mikroserwisowych.

SQL Server / PostgreSQL / MongoDB

Praca z relacyjnymi i nierelacyjnymi bazami danych: SQL Server, PostgreSQL, MySQL, MongoDB i CosmosDB. Modelowanie danych, optymalizacja wydajności i migracje.

Docker

Konteneryzacja aplikacji .NET i Node.js z użyciem Dockera. Doświadczenie z multi-stage builds, Docker Compose i optymalizacją obrazów.

Kubernetes

Wdrażanie i zarządzanie mikroserwisami w klastrach Kubernetes. Znajomość Helm, CRD, autoskalowania i zarządzania zasobami.

Azure Functions

Tworzenie aplikacji serverless z wykorzystaniem Azure Functions. Integracja z Event Grid, Service Bus i innymi usługami natywnymi Azure.

Redis

Wdrożenie rozproszonego cache’owania za pomocą Redis w aplikacjach .NET o dużym obciążeniu. Doświadczenie z politykami wygasania, pub/sub i trwałością danych.

RabbitMQ

Użycie RabbitMQ do budowy architektur opartych na zdarzeniach i komunikacji asynchronicznej między mikroserwisami. Doświadczenie z konfiguracją kolejek, exchange’ów, dead-letteringiem i wzorcami konsumentów.

Azure Service Bus

Integracja z Azure Service Bus do niezawodnej komunikacji między rozproszonymi usługami .NET. Doświadczenie z kolejkami, tematami, subskrypcjami i strategiami retry/dead-lettering.

Git / Azure DevOps / Bitbucket

Praca z Git w zespołach developerskich. Tworzenie i utrzymywanie pipeline’ów CI/CD z użyciem Azure DevOps i Bitbucket Pipelines.

Grafana / Splunk

Konfiguracja dashboardów monitorujących z wykorzystaniem Grafana i Splunk. Agregacja logów z usług .NET i konfiguracja alertów.

Angular

Tworzenie responsywnych interfejsów użytkownika w Angularze. Praca z TypeScriptem, HTML i CSS oraz integracja z backendem .NET.

Blazor

Rozwój interaktywnych interfejsów webowych z użyciem Blazor WebAssembly i Blazor Server. Integracja z API .NET i współdzielenie modeli domenowych.

Node.js

Tworzenie lekkich usług backendowych i narzędzi CLI w Node.js. Znajomość Express i narzędzi z ekosystemu.

ASP.NET

Tworzenie solidnych API REST, mikroserwisów i aplikacji fullstack w ASP.NET Core i ASP.NET MVC. Doświadczenie z middleware, uwierzytelnianiem, integracją EF i warstwową architekturą aplikacji.

Prometheus

Integracja Prometheus z mikroserwisami .NET do zbierania metryk i monitorowania. Znajomość PromQL, AlertManagera i tworzenia własnych eksporterów.
